## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run d:start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# integration tests
$ yarn run d:test:int
$ yarn run d:test:int:watch

# test coverage
$ yarn run d:test:int:cov
```
